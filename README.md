# Cheerful Givers Community Outreach

A simple responsive website built using HTML, CSS, and JavaScript for the Cheerful Givers community initiative in Bloekombos, Kraaifontein.

The website presents the organisation’s mission, services, team, and contact details in a clean landing page format.

---

## Features

- Full-screen section-based layout
- About section with image and text layout
- Services displayed using cards
- Team section with profile cards
- Contact section with embedded Google Map
- Scroll reveal animations
- Responsive design for mobile and desktop

---

## Technologies Used

- HTML5
- CSS3 (Flexbox and Grid)
- JavaScript (Vanilla)

---

## Responsive Design

- Desktop: multi-column layout
- Mobile: single-column stacked layout

---

## Purpose

This project was created as a school assignment to demonstrate frontend development skills, responsive design, and basic JavaScript interactivity.
